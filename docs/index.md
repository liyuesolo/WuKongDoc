# WuKong Simulation

WuKong is a light-weight simulation codebase for different type of discretizations commonly used in computer graphics.

WuKong supports

=== "Discrete Shell"
     **Discrete Shell**, see the [Discrete Shell Documentation](./projects/discrete-shell.md).
